# who-am-i 프로젝트

[화면보기](https://azurealstn.github.io/who-am-i/)

## ❕ 설명

- 취준생의 하루 일과를 나타난 사이트입니다.

## 🕳 기술 스택

- HTML, CSS, JavaScript

## ✔ 기능

- Interactive하게 스크롤을 이용한 페이지 구성
- IntersectionOberserver API를 이용하여 관찰 범위를 정하여 그 범위만큼 루프
